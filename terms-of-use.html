23. ABOUT THE APP

Request to know the status (open or closed) or information about any place (stores, restaurants, temples, clinics, etc), anytime, anywhere with the "anyone_there?" app.

Sometimes people can't find the status of a place online and sometimes there might not be a phone number available to get more information about a location. Well, not anymore. For example, the status of a store or a restaurant might be displayed as open online, but in reality, the restaurant might be closed. In cases like this, if someone wants to know if a store or a restaurant is actually open, or if they want to know if a particular item is available somewhere, all they have to do is just place a request on the anyone_there app providing sufficient details, and someone near the location can volunteer to help.

Without making a trip, with the anyone_there app at your fingertips, it's now so easy to get any kind of information about a place. The app will completely bypass the hassle of you driving all the way to places where the commute might be impractical.

24. HOW TO USE THE 'ANYONE_THERE' APP

1. Login:

Once the user installs the 'anyone_there' app, the user has to provide their name and their mobile number to get an OTP. The user can login to the

anyone _there' app using the OTP.

2. Request for information about any place:

After logging in, the user can select a location on the map by typing in the address or by pinpointing a location manually on the map and then, provide details of what information the user is looking for. Once a request is placed, the 'anyone_there' app will look for people who can help provide the information that the user is looking for.

If there's no one available to provide the user with the information they require, then the app will say so.

If someone responds to the users request, then their name and mobile number will be available to the user so that the responder can be contacted by the user to get the required information. Responder may charge a reasonable fee based on the users (requester) consent to provide the information. It is purely up-to the discretion of the requester to decide how much money (Suggestion: Requestor may use distance needed to travel to the requested place as a parameter to calculate a reasonable amount) they are willing to pay to get the information that they require and they both can decide on the means of completing the transaction outside the anyone_there app. Because, it is not in the anyone_there apps scope to assist with or verify the transaction. Hence, the anyone_there app or its publisher (or app owner) is not liable for this transaction in anyway and the users of anyone_there app should not hold anyone_there app publisher (or owner) accountable for these transactions.

3. Help Others:
Similarly, users can help others who need information about a place in the users vicinity. To do this, the user can click the Help Others button and check if there are any requests and respond to them. Once the user responds to a request, the requester receives the responders name and their mobile number using which the requester can reach out to get more information about the place. This will also serve the user an opportunity to make money. See section #2 above to understand what amount to charge or expect from the requester to provide them the information.

4. Charges:

While using the 'anyone_there' app, users can place up to 5 requests for free and subsequent requests can be availed by paying nominal charges. Users have to pay a nominal fee before placing a request. If for any reason after placing a request, a user doesn't get any response within the stipulated time, then the paid amount will be stored as credit in the user's account for future use. This amount can be utilized for any new requests.
